assumption:
Each transaction invloves only a single book


Entities (Models)
-------------
BOOK
STUDENT
STUDENT ACCOUNT
AUTHOR
TRANSACTION

Functionalities (Controllers)
----------------
CRUD for BOOK
CRUD for STUDENT
CRU for TRANSACTION
